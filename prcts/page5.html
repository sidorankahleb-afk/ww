Позиционирование в CSS

Все типы позиционирования

position: relative;

position: absolute;
Относительно родителя

Родительский контейнер с position: relative

Значение Описание
static Обычный поток документа
relative Смещение относительно своего места
absolute Относительно ближайшего позиционированного предка
fixed Относительно окна браузера

float используется для обтекания (см. страницу 4).
